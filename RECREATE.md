## Vue app

Create the project

```
vue create kubezico
```

There are only 3 dependencies : Vuetify JS, axios, vue json pretty

```
vue add vuetify
yarn add axios
yarn add vue-json-pretty@1.7.1
```

Important to fix the vue json pretty version

Install the plugin
```
vue add electron-builder
```
electron version 9

Fix the icons and procedure
